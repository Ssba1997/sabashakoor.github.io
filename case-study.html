🚲 Citi Bike Dashboard

Tools Used: Python, Streamlit, Tableau, NOAA API

🧩 Project Overview

This dashboard explores bike usage patterns in NYC during 2022. The goal was to identify seasonal trends, popular start stations, and optimize bike distribution strategy using weather data integration.

🔧 My Role

I cleaned and merged Citi Bike and NOAA weather datasets, built visualizations in Streamlit, and embedded a Kepler map for spatial insights.

📊 Key Results

  • 🏙️ Identified busiest bike corridors across Manhattan and Brooklyn.
  • 📈 Found peak usage during May–October, with drops in winter months.
  • 🚲 Suggested 25–30% more bike stock in peak seasons.
